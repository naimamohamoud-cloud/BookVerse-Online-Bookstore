# BookVerse Online Bookstore

## Project Overview

BookVerse is a responsive online bookstore website designed to help readers discover, explore, and learn about books across multiple categories. The website provides information about featured books, authors, best sellers, and book categories through a clean and user-friendly interface.

## Objectives

* Create a fully responsive multi-page website.
* Apply HTML5, CSS3, JavaScript, and Bootstrap.
* Provide realistic bookstore content.
* Implement interactive JavaScript features.
* Demonstrate effective use of Git and GitHub version control.

## Planned Pages

1. Home
2. About Us
3. Books
4. Categories
5. Best Sellers
6. Authors
7. Contact Us
8. FAQ

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Git & GitHub
